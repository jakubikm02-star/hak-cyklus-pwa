# HAK Cyklus PWA

**28-dňový cyklus tracker** pre sledovanie účinnosti hormonálnej antikoncepcie (HAK).

Jednoduchá Progressive Web App (PWA) s:

- Kalendárom 28-dňového cyklu
- Označovaním vynechaných dňov (kliknutím na deň)
- Automatickým zvýraznením **7-dňového rizikového okna** po vynechaní
- Farebným označením fáz: not safe (1-6, 25-28), safe (7-20), prechod (21-24)
- Status banner s aktuálnym dňom a odporúčaním
- Export / import zálohy (JSON)
- Reset na dnešok ako deň 1
- Plná offline podpora (Service Worker + PWA manifest)
- Inštalovateľná na mobil (Add to Home Screen)

## Použitie
1. Nastav deň 1 cyklu (začiatok balenia alebo posledný štart)
2. Označuj vynechané tabletky kliknutím na deň v kalendári
3. Sleduj rizikové okno (7 dní po vynechaní) – vždy použi doplnkovú ochranu

**Dôležité:** Toto je pomocný nástroj, nie náhrada lekárskej rady. Vždy konzultuj s gynekológom pri pochybnostiach o účinnosti.

Vytvorené v kontexte presného sledovania 7-dňového okna po vynechaní HAK.
